Pour compiler complètement le manuel dans les formats :
- html : installez "pandoc"
- pdf et html : vous devrez installer une distribution Latex (par ex. texlive).
Vous devrez aussi vous assurer que les extensions appelées dans le fichier :
- 00-grisbi-manual-$LANG.tex,
par la commande \usepackage{package}, sont installées et en particulier :
(pour vérifier si l'extension est présente, faire la commande suivante : kpsewhich -all "package".sty)
[si seulement "package".dtx présent, taper en ligne de commande: << tex package.dtx >> ou << tex package.ins >>
pour générer package.sty]

[pour installer, copier dans le:
- home: ~/texmf/tex/latex/"package"/
ou
- systeme: /usr/share/texlive/texmf-dist/tex/latex/"package"/]

La compilation se fait avec les commandes suivantes, dans le répertoire src/3.0/$LANG :

# compile au format PDF avec images, index et glossaire :
$ make pdf

# compile au format HTML :
$ make html_pandoc


Pour nettoyer le répertoire src/ en supprimant TOUS les fichiers
auxiliaires créés par une compilation précédente (.aux, .log, etc.) :
$ make clean

Ou bien supprimer ces mêmes fichiers et TOUS les fichiers créés par le
Makefile (c'est ce qui est recommandé de faire avant chaque compilation) :
$ make distclean

Après compilation:
- le fichier grisbi-manuel_3.0-$LANG.pdf se trouve dans src/3.0/$LANG/pdf/
- le fichier grisbi-manuel_3.0-$LANG.html se trouve dans src/3.0/$LANG/html/
